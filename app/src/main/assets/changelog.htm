MAME4droid 变更日志

发展：
  • 2020-09-07 版本 1.15
    Android 10.0 支持。为超宽手机添加了默认横向布局。改进的纵向触摸控制器。改进了对直接从文件浏览器打开压缩 rom 的支持。使用最新 NDK、clang 和 libc++ 的新版本。一些错误修复。
  • 2020-01-25 版本 1.14
    安卓 9.0。Xbox One 蓝牙控制器原生支持。修复。
  • 2019-05-26 版本 1.13
    Android 8.0 和 64 位支持。
  • 2017-01-22 版本 1.12
    修复了 Android Nougat 和新的 Shield TV。
  • 2016-06-06 1.11 版
    增加了通用多控制器支持。
  • 2015-10-26 版本 1.10
    添加了 Android 6.0 (Marshmallow) 支持。
  • 2015-04-18 版本 1.9.3
    增加了 Android TV 支持。
  • 2015-03-20 1.9 版本
    为 Nvidia Shield 设备添加了鼠标支持（+轨迹球，如 X-Arcade Tankstick）（您必须在设置中启用）。新的视频同步和刷新速度选项。改进的触控 DPad。新的触摸按钮。
  • 2015-01-18 1.8版
    增加了可选择的HQx图像增强效果。添加了触摸光枪自动检测。添加高级设置以更改默认安装路径。添加了一个新的自定义 BIOS 设置以在 Neogeo 上设置像 unibios 这样的备用 BIOS。添加了新的整数缩放选项。Shane R. Monroe 的新帮助。修复了一些严重的错误。
  • 2015-01-07 1.7 版
    增加了棒棒糖设备的材料设计。改进了对 >= kitkat 设备的 sdcard 支持。添加了直接打开 ROM 文件的意图。改进的叠加（扫描线）渲染。添加了可选的真彩色渲染。改进了对 NVidia shield 平板电脑和控制器的支持。一些错误修复。
  • 2014-01-10 版本 1.6.1
    JXD S7800b，Archos Gamepad 2 支持。改进的沉浸式模式。添加了删除 nvram 和 cfg 文件的选项。添加了可选的保存加载组合。一些错误修复。
  • 2013-12-10 版本 1.6.0
    添加了点对点网络多人游戏。添加了 kitkat 沉浸式模式。
  • 2013-11-10 版本 1.5.3
    在功能强大的设备上，某些 3d 游戏（例如铁拳）的速度提高了 30%。修复了一些游戏，例如 brapboys。添加了过扫描选项。修复了大力水手等游戏的美术错误。
  • 2013-11-03 版本 1.5.1
    NVidia Shield 和 Ouya 改进和错误修复。添加了新的雇佣模式（现在 rom manager 可以在雇佣分辨率下呈现）。固定控制器进入睡眠错误。
  • 2013-10-06 1.5 版本
    增加了对欧雅的支持。添加了在 NVidia Shield 上修复垂直倾斜的选项。添加了更多缩放选项，现在可以超出屏幕边界。将选择按钮更改为“X”按钮以与 Android 默认按钮保持一致。一些错误修复。
  • 2013-08-20
    版本 1.4.4 在支持的设备上添加了 OpenSL 低延迟音频。
  • 2013-08-17 版本 1.4.3
    添加了新的叠加效果（带有叠加强度选择器）。添加了果冻豆熄灯模式（或在触摸控制器隐藏时隐藏导航栏）。新的像素方面智能选项可最大化仿真空间。添加了棒/DPAD 触摸大小选项。添加了更多缩放模式。添加了在 GamePad 上禁用右摇杆的选项。PS3 游戏手柄自动检测的一些错误修复。
  • 2013-08-12 版本 1.4.2
    NVidia Shield 自动配置修复。
  • 2013-08-03 版本 1.4.1
    添加了游戏过滤器（克隆、年份、类别、制造商、驱动程序源...）。一些错误修复。
  • 2013-07-27 版本 1.4。
    在触摸屏上添加了轻触作为光枪设备。添加了倾斜传感器选项以点击屏幕上的任意位置进行射击。一些错误修复。
  • 2013-07-19 版本 1.3.5
    一些错误修复。
  • 2013-07-18 版本 1.3.4
    改进了倾斜传感器的精度，现在具有向上/向下功能。一些错误修复。
  • 2013-07-14 1.3.3 版本
    增加了新的纵向全屏模式。一些错误修复。
  • 2013-07-10 版本 1.3.2
    调整了默认触摸布局。一些错误修复。
  • 2013-07-08 版本 1.3.1
    一些错误修复。
  • 2013-07-04 版本 1.3
    添加了 NVIDIA Shield 支持。增加了许多 USB / 蓝牙游戏手柄的即插即用检测。为检测到的游戏手柄添加了 ICS 完整游戏控制器支持（多个控制器、模拟输入和双摇杆）。添加了 hiscores 保存（MKChamp 补丁）。在某些游戏中添加了强制刷新率开关以实现更流畅的游戏玩法。添加了自动射击。修复了 SEGA 和 CAVE 驱动程序上的一些匿名计时器，以修复保存状态问题（AWJ 补丁）。添加了收藏夹。在 ROM 管理器中添加了删除游戏的选项。添加了矢量默认选项。增加了仿真速度。添加了 1-6 个按钮和 2-8 种方式的自动选择。改进的 ROM 管理器。很多错误修复。
  • 2012-06-08 版本 1.2.1
    添加了双棒 ROM（如机器人）的默认按钮映射。添加了可选择的暂停通知。一些错误修复。
  • 2012-05-08 版本 1.2。
    添加了本地多人游戏（最多 4 名玩家）。添加了真正的模拟控制（它可以制作更多可玩的模拟游戏，如 StarWars 和光枪游戏，如 owolf）。修复了 Taito X 系统（现在超人工作）。添加了 P1 播放器作为 P2、P3、P4 输入选项。一些错误修复。
  • 2012-04-15 版本 1.1.2。
    添加了触摸按钮大小选择器，修复了工件\改进了自动分辨率模式的缩放，修复了一些 Opengl 问题。
  • 2012-04-05 版本 1.1.1
    升级到 MAME 0.139u1 以修复使用 OKI 音频的驱动程序的保存状态问题。启用 SH2 核心，现在可以使用 CPS3。添加了速度技巧以制作可玩的相同游戏，可通过 MAME 菜单在游戏中按硬币+开始访问。添加了强制像素纵横比选项。一些错误修复。
  • 2012-03-31 版本 1.1。
    升级到 MAME 0.139。在 MAME 菜单中修复/添加了输入映射。添加了正确的 4/3 纵横比（现在可以使用 MAME 菜单纵横比选择器）。修复了 GL 模式下的闪烁问题。改进的 ROM 管理器（仍然有待处理的东西）。添加了模拟分辨率选择器以改进图稿渲染。
  • 2012-02-27 版本 1.0
    WIP。第一个版本。
