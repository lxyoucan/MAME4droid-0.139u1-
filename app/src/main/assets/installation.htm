安装

首次安装

首次运行 MAME4droid 时，它会在您的 Android 设备的内存中为您创建一个文件夹结构。

默认文件夹为： /sdcard/MAME4droid/（查看 2020 年注）

此文件夹包含 MAME 使用的所有其他文件夹以及一些基本配置文件。

由于 MAME4droid 不附带游戏 ROM 文件，您需要自己将它们复制到/sdcard/MAME4droid/roms文件夹中。这些应正确命名，压缩的 MAME v1.39u1 ROM 文件，文件名全部小写。

2020 注意：从 Android 10 开始，Google 增加了范围存储。这意味着应用程序不能再读取 SD 卡上的任何位置，只能读取它们自己的目录。我使用了一个兼容性标志，它允许应用程序像以前一样工作，但它可以随时停止工作。这就是为什么现在应用程序将安装文件并在/sdcard/Android/data/com.seleuco.mame4droid/files中创建 roms 目录，而不是旧的/sdcard/MAME4droid/路径。这样做的缺点是，当您卸载应用程序时，所有文件（例如 rom 或保存状态）都将被删除。很好，我不再需要从您的 SD 中读取的权限。但是，只要它有效，您就可以继续使用旧方式。

更新/自定义安装

由于 Android 操作系统的 KitKat 和 Lollipop 版本增强了 SD 卡安全性，MAME4droid (0.l39u1) 版本 1.7（2015 年 1 月 7 日）对默认位置和安装进行了一些更改。这可能会对以前安装的 MAME4droid 用户产生不利影响，同时帮助那些使用这些版本的 Android 的用户。

您的 ROM 仍然安全可靠 - 就在您离开它们的地方，可能在您的外部 microSD 卡上。快速解决方法是简单地进入Options->Settings->General->Change ROMs Path。将此指向您的实际 ROM 文件夹（而不是您的 MAME4droid 文件夹）。如果您没有自定义过滤器或打开了“收藏夹”，您现在应该可以看到所有记忆中的 ROM。

即使更改 ROM 路径，您也可能会丢失收藏夹等配置设置。一个修复？将所有文件（不是文件夹）从 MAME4droid 文件夹（mame4droid.cfg、收藏夹.ini 等）的根目录复制到 / /sdcard/MAME4droid。覆盖已经存在的任何文件。重新启动 MAME4droid。

如果您是高级用户，您可以将整个安装（不仅仅是 ROM 文件夹）重新定位到另一个位置（就像您之前的位置一样）。您将需要安装的完整路径。您可以在Options->Settings->Advanced->Installation Path下更改它。小心使用此选项；如果您选择了有效但不正确的路径，则会在那里进行 MAME4droid 的全新安装。仅推荐给有经验的用户。

文件夹配置

MAME4droid 可以利用许多高级 MAME 功能，例如边框和选取框艺术品和声音样本（对于需要它们的游戏）。将适当的资产放入相应的文件夹中。查看资源和资产部分以获取资产资源。

Artwork/ -> Artwork 目录
cfg/ -> MAME 配置文件目录
hi/ -> Hiscores 目录
nvram/ -> NVRAM files 目录
roms/ -> ROMs 目录
samples/ -> Samples 目录
snap/ -> 屏幕快照目录
sta/ ->保存状态目录
