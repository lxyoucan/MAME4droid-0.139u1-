控件

这些是默认设置，如果您更改控制器映射，可能并不总是适用。

虚拟垫

  • 屏幕控制器和按钮
  • 模拟控制鼠标
  • Lightgun（点击屏幕；必须在Options->Settings->Input->Touch Lightgun下启用）

游戏手柄

  • 控制器上的按钮 A、B、C、D、E、F 映射到按钮 1 到 6
  • 选择按钮插入硬币/添加信用
  • START键开始1P游戏

特别的

  • START+UP 开始 2P 游戏。START+RIGHT 开始 3P 游戏。START+DOWN 开始4P游戏
  • SELECT+UP 插入 2P 积分。SELECT+RIGHT 插入 3P 积分。SELECT+DOWN 插入 4P 积分
  • R1 + START 加载一个保存状态
  • L1 + START 保存一个保存状态
  • L2 退出选择菜单以选择另一个游戏
  • START + SELECT 访问游戏的 MAME 菜单（dip 开关等）
  • R2 打开帮助和设置菜单

注意：要在 MAME 需要时键入 OK，请按 LEFT，然后按 RIGHT。
